# 로또 번호 예측 생성기 (Lotto Predictor)

## Project Overview
AI 가중치 알고리즘을 활용하여 로또 번호를 생성하는 모던 웹 애플리케이션입니다. 사용자에게 세련된 UI와 애니메이션 효과를 통해 행운의 번호를 제공합니다.

## Technical Stack
- **HTML5**: 시맨틱 구조 및 레이아웃
- **CSS3**: Modern Baseline features (CSS Variables, Flexbox, Animations, Glassmorphism)
- **JavaScript**: ES Modules, 가중치 기반 랜덤 추출 알고리즘

## Features & Design
- **AI 가중치 알고리즘**: 단순 랜덤이 아닌, 내부 가중치(가짜 통계 기반)를 활용한 번호 추출.
- **모던 디자인**: 
  - Dark Mode 테마 (Slate/Blue/Green 조합)
  - 로또 번호 대역별 자동 색상 지정 (1-10 노랑, 11-20 파랑 등)
  - Staggered Animations (번호가 순차적으로 나타나는 효과)
  - Glassmorphism 카드 레이아웃
- **반응형 디자인**: 모바일 및 데스크탑 최적화.
