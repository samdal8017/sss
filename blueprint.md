# 로또 번호 예측 생성기 (Lotto Predictor)

## Project Overview
AI 가중치 알고리즘을 활용하여 로또 번호를 생성하는 프리미엄 웹 애플리케이션입니다. 사용자에게 세련된 UI와 애니메이션 효과를 통해 행운의 번호를 제공하며, 모바일과 PC 모두에 최적화되어 있습니다.

## Technical Stack
- **HTML5**: 시맨틱 레이아웃 및 웹 구조
- **CSS3**: Modern Baseline features (Flexbox, Grid, CSS Variables, Animations, Glassmorphism)
- **JavaScript**: 가중치 기반 랜덤 추출 알고리즘, 반응형 인터랙션

## Features & Design
- **AI 가중치 알고리즘**: 가중 랜덤 추출을 통해 생성된 행운의 번호 세트 제공.
- **모던 프리미엄 디자인**: 
  - Dark Mode & Glassmorphism 테마 적용
  - 로또 번호 대역별 자동 색상 테마 (노랑, 파랑, 빨강, 회색, 초록)
  - Staggered & Pop-in 애니메이션 효과
- **완벽한 반응형 (PC & Mobile)**: 
  - PC: 멀티 컬럼 그리드 레이아웃
  - Mobile: 스택 레이아웃 및 터치 최적화 버튼
- **추가 기능**:
  - **당첨결과 확인**: 동행복권 공식 사이트로 연결되는 골드 테마 버튼.
  - **제작자 표기**: "Designed & Developed by Kim Do-wan" 푸터 포함.
